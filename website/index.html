About the Project

CipherX is a Java-based 2FA authentication system designed as a secure login solution using OTP verification. It focuses on strong authentication, user validation, and activity logging to enhance system security.

Features

  • User Registration & Secure Login
  • OTP Generation & Validation
  • Centralized Log Tracking
  • Simple, lightweight code for fast authentication
